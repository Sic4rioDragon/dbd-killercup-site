About

This is the page for the current DBD Killer Cup test run. The idea is pretty simple: use the spreadsheet for a killer-based tournament format, and if people want to do their own version, they can copy the main template and edit it themselves.

How it works

1) Killer Pool

  • Set Have? to Have or Don't have for each killer.
  • Auto-fill is off by default.
  • Turn it on when you want the Main tab to fill in.

2) Main

  • Each group has 5 killers.
  • Group stage uses a 5-round round-robin with no rematches.
  • Each round has 2 matches and 1 bye.
  • Enter hooks from 0–12 in the Hooks cells.

3) Points

  • Win (more hooks) = 3 points
  • Draw (equal hooks) = 1 point each
  • Loss = 0 points

4) Tiebreaker

  • The extra Tiebreaker column is only used if an additional 1v1 is needed to decide the group winner.
  • Bye means that killer does not need to fight that round.

Want to run your own?

If you want to do your own Killer Cup, use the main template spreadsheet and make your own copy before editing it. That way the original stays untouched and you can change whatever you want.
